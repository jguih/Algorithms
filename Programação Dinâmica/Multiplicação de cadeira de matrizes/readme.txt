O programa retorna uma matriz com o custo computacional de cada operação e o menor custo

Exemplo:
Entrada: Quantidade de matrizes: 4; Dimensões: 5, 4, 6, 2, 7;
Saída: 
     M  1   2   3    4   
     1 [0][120][88][158]
     2  [  ][0][48][104]
     3   [  ][  ][0][84]
     4   [  ][  ][  ][0]
Menor custo para multiplicar as 4 matrizes: 158
